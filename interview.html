Ques1:Difference between flex row and column

flex-direction: row (Default) Main axis: Horizontal (left to right). Cross axis: Vertical (top to bottom). Effect: Flex items are laid out in a row, side by side.

flex_row

In CSS Flexbox, the flex-direction: column property sets the main axis to vertical. This means that flex items are arranged from top to bottom inside the flex container.

Ques4:Difference between absolute and relative positioning

position: relative The element is positioned relative to its normal position in the document flow. It still takes up space in the layout (others don't shift). You can use top, right, bottom, and left to move it slightly from where it would normally be. 
.relative-box {
position: relative;
top: 10px;
left: 20px;
}

position: absolute The element is removed from the normal document flow (other elements behave as if it doesn’t exist). It is positioned relative to the nearest positioned ancestor (an element with position: relative, absolute, or fixed). If no such ancestor exists, it is positioned relative to the (the document body). 
.absolute-box {
position: absolute;
top: 0;
right: 0;
}

Ques6:What is the difference between padding and margin?

Padding: Space inside an element, between its content and border. It affects the element’s size

Margin: Space outside an element, between its border and other elements. It does not affect the element’s size.
